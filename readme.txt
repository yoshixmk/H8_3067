付属CD-ROMの内容と使用上の注意点

付属CD-ROMには、H8/3067スタータキットのプログラム開発環境と、「H8マイコンによるネットワーク・プログラミング（青木直史著、技術評論社、2008年発行）」の各章で説明したプログラムを収録しています。

付属CD-ROMの各フォルダを「マイコンピュータ」→「ローカルディスク(C:)」のなかにコピーし、ご利用ください。

各フォルダの内容はつぎの通りです。

--
・chapter3
　第3章で説明したプログラムが収録されています。

・chapter4
　第4章で説明したプログラムが収録されています。

・chapter5
　第5章で説明したプログラムが収録されています。

・chapter6
　第6章で説明したプログラムが収録されています。

・appendix
　Appendixで説明したWindows PCのプログラムが収録されています。

・SAMPLE
　製品出荷時に書き込まれているプログラムが収録されています。

・MANUAL
　H8/3067スタータキットの取扱説明書とLCD資料が収録されています。

--

付属CD-ROMのプログラムは著作権法により保護されています。付属CD-ROMのプログラムの一部または全てを無断で転載することを禁止します。

付属CD-ROMのプログラムを利用することにより発生した損害などに関して、著作権者である青木直史および(株)北斗電子は責任を負いかねますのでご了承ください。
